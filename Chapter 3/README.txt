This chapter demonstrates Arduino Raspberry Pi interface. The code samples include demonstrate a Weasley weather clock using an Arduino or Raspberry Pi.  

Dependencies: 

* Blinkm Arduino Library - https://code.google.com/p/blinkm-projects/downloads/list
* Adafruit_GFX library - https://github.com/adafruit/Adafruit-GFX-Library
* Adafruit_LEDBackpack library - https://github.com/adafruit/Adafruit-LED-Backpack-Library
